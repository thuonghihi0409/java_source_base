# KẾ HOẠCH HỌC JAVA SPRING BOOT

## 🎯 Mục tiêu

Chuẩn bị kiến thức vững chắc về Java Spring Boot để thi lập trình vào vị trí lập trình Java Spring Boot.

## 📚 Lộ trình học tập (4-6 tuần)

### **TUẦN 1: Nền tảng Spring Boot**

**Mục tiêu:** Hiểu cơ bản về Spring Boot và cách tạo ứng dụng đầu tiên

#### Ngày 1-2: Giới thiệu Spring Boot

- [ ] Spring Boot là gì? Tại sao sử dụng Spring Boot?
- [ ] Spring Boot vs Spring Framework
- [ ] Auto-configuration và Starter dependencies
- [ ] Spring Boot CLI và Spring Initializr

#### Ngày 3-4: Tạo project đầu tiên

- [ ] Sử dụng Spring Initializr (start.spring.io)
- [ ] Cấu trúc project Spring Boot
- [ ] @SpringBootApplication annotation
- [ ] Application.properties và application.yml
- [ ] Chạy ứng dụng Spring Boot

#### Ngày 5-7: Dependency Injection và IoC

- [ ] @Component, @Service, @Repository, @Controller
- [ ] @Autowired và @Qualifier
- [ ] @Configuration và @Bean
- [ ] @Primary và @Conditional
- [ ] Profile và Environment

### **TUẦN 2: Web Development với Spring Boot**

**Mục tiêu:** Xây dựng RESTful API cơ bản

#### Ngày 8-10: Spring MVC

- [ ] @RestController và @Controller
- [ ] @RequestMapping, @GetMapping, @PostMapping
- [ ] @PathVariable và @RequestParam
- [ ] @RequestBody và @ResponseBody
- [ ] Exception Handling với @ControllerAdvice

#### Ngày 11-12: Data Binding và Validation

- [ ] @Valid và validation annotations
- [ ] Error handling và error messages
- [ ] Custom validators
- [ ] Content negotiation (JSON, XML)

#### Ngày 13-14: Thực hành

- [ ] Tạo RESTful API hoàn chỉnh
- [ ] CRUD operations
- [ ] Testing với Postman

### **TUẦN 3: Database và JPA**

**Mục tiêu:** Làm việc với cơ sở dữ liệu

#### Ngày 15-17: Spring Data JPA

- [ ] JPA và Hibernate
- [ ] @Entity, @Table, @Id, @GeneratedValue
- [ ] @Column, @JoinColumn, @OneToMany, @ManyToOne
- [ ] Repository pattern
- [ ] JpaRepository và custom queries

#### Ngày 18-19: Database Configuration

- [ ] H2 Database (in-memory)
- [ ] MySQL/PostgreSQL configuration
- [ ] Connection pooling
- [ ] Database migration với Flyway/Liquibase

#### Ngày 20-21: Advanced JPA

- [ ] @Query và @NamedQuery
- [ ] Pagination và Sorting
- [ ] @Transactional
- [ ] Lazy vs Eager loading

### **TUẦN 4: Security và Testing**

**Mục tiêu:** Bảo mật và kiểm thử ứng dụng

#### Ngày 22-24: Spring Security

- [ ] Authentication và Authorization
- [ ] JWT (JSON Web Token)
- [ ] CORS configuration
- [ ] Password encoding
- [ ] Role-based access control

#### Ngày 25-26: Testing

- [ ] Unit testing với JUnit 5
- [ ] Integration testing
- [ ] @SpringBootTest
- [ ] MockMvc testing
- [ ] TestContainers

#### Ngày 27-28: Thực hành tổng hợp

- [ ] Xây dựng ứng dụng hoàn chỉnh
- [ ] API documentation với Swagger
- [ ] Logging và monitoring

### **TUẦN 5-6: Nâng cao và Chuẩn bị thi**

**Mục tiêu:** Kiến thức nâng cao và luyện thi

#### Tuần 5: Advanced Topics

- [ ] Microservices với Spring Boot
- [ ] Spring Cloud (Service Discovery, Config Server)
- [ ] Message queues (RabbitMQ, Kafka)
- [ ] Caching với Redis
- [ ] Actuator và Health checks

#### Tuần 6: Luyện thi

- [ ] Ôn tập tất cả kiến thức
- [ ] Làm bài tập thực hành
- [ ] Mock interview
- [ ] Chuẩn bị portfolio projects
